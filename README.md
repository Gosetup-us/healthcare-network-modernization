# Healthcare Network Modernization
Enterprise-grade Small Healthcare Network built with OPNsense, Proxmox VE, Unifi following Enterprise Best Practices.

## Project Overview
The Healthcare Network Modernization Project is a real-world enterprise infrastructure initiative designed to modernize the IT environment of a small healthcare facility in the United States.

The project focuses on designing, building, testing, documenting, and deploying a secure, scalable, and maintainable network infrastructure based on enterprise best practices.

The complete solution is first developed and validated in a home laboratory before being deployed into the production environment. This approach minimizes deployment risks while ensuring proper documentation, repeatability, and operational reliability.

Core technologies include OPNsense, Proxmox VE, UniFi networking, VLAN segmentation, virtualization, centralized management, and network security.

This repository documents every stage of the project, including architecture design, implementation, configuration, testing, troubleshooting, and final deployment.
